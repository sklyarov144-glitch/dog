# Surf Dog Rush — MVP для Yandex Games

Готовый HTML5 Canvas MVP браузерного раннера: щенок-серфер, 3 дорожки, препятствия, монеты, магазин скинов, сохранение и заготовки под Yandex Games SDK.

## Запуск локально

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Управление

- Свайп влево/вправо — смена дорожки.
- Свайп вверх или тап — прыжок.
- Свайп вниз — подкат.
- Клавиатура: стрелки, WASD, пробел.

## Интеграция с Yandex Games

В `index.html` уже подключен SDK:

```html
<script src="https://yandex.ru/games/sdk/v2"></script>
```

В `game.js` уже есть заготовки:

- `YaGames.init()`
- `ysdk.features.LoadingAPI.ready()`
- `ysdk.adv.showFullscreenAdv()`
- `ysdk.adv.showRewardedVideo()`
- `ysdk.leaderboards.setLeaderboardScore('surfDogRushScore', score)`

Перед публикацией в консоли Yandex Games создайте лидерборд с техническим именем:

```text
surfDogRushScore
```

## Юридическая заметка

Не используйте название Subway Surfers, их персонажей, уровни, интерфейс или узнаваемые элементы. Фото щенка лучше использовать как референс, а не как финальный игровой ассет, если нет подтверждённых прав.
